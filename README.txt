Teamkill BF3 plugin for Big Brother Bot (www.bigbrotherbot.net)
===============================================================

By Courgette


Description
-----------

This plugin kills any player killing a team mate on a BF3 server



Installation
------------

 * copy tkbf3py into b3/extplugins
 * update your main b3 config file with :

<plugin name="tkbf3" />
